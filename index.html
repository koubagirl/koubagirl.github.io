Predicting Diabetes Through Machine Learning


Disease progression and chronic disease progression are something very important to me. A few different projects I’ve worked on involve “predicting” diabetes. This project involves using multiple symptoms upon intake to a hospital to accurately predict through machine learning whether a patient is suffering from undiagnosed diabetes. In 2022, the CDC estimated that there are roughly 8.5 million Americans suffering from diabetes unknowingly. Given the damage diabetes can do, this is a sobering figure and more needs to be done to change this statistic. My Random Forest model was able to accurately predict diabetes 99.4% of the time.

Other Keras Deep Learning Projects


This "project" is comprised of three similar projects all using Keras deep learning to create convolutional neutral networks. This project focuses on three different datasets, namely IMDB movie reviews, Reuters news reports, and the Keras Boston housing datasets, each showcasing different techniques for building neural networks. First, the simplest of the three projects, is a binary classification where the movie reviews are classified as positive or negative, based on the text content of the reviews. Second, slightly more advanced, is a multiclass classification example, where short newswires are classified into 46 topics based on their text content. Third is the Boston housing price dataset which uses features of different scales to predict the median price of homes in a Boston suburb.
